# Consolata University Site

Welcome to the Consolata University Site repository.

## Site Purpose

This site powers [ciu.ac.ke](https://ciu.ac.ke), the official portal for Consolata University. It provides resources, information, and tools for students, staff, and visitors.

## Getting Started

- **Access the Site**: Visit [ciu.ac.ke](https://ciu.ac.ke) to view and use all available features.
- **Requirements**: Most features are public, but some may require registration or authentication for access.

## Key Features

- **Admissions Display**: Information on current student admissions.
- **Timetables**: View class or staff timetables.
- **Administration Portal**: Management for university admin staff.
- **Student Section**: News, resources, and student support.
- **Authentication**: Login for users with privileged site access.

Explore sections such as:
- `consolata_admin/` for admin features.
- `home/`, `student/`, `staff_teachers/` for role-specific resources.
- `core/`, `userauth/` for main logic and authentication flows.

## How to Use

- Browse ciu.ac.ke to locate the resources you need.
- Admin, teachers, and students have specific sections with tailored views.
- For help, use the contact or support information provided on the site.

## Getting Help

- For technical issues or questions, contact site admins through the support channels listed on [ciu.ac.ke](https://ciu.ac.ke).

## Repository

This repository contains all code and resources for the university site. It depends on packages listed in `requirements.txt` and organizes its core logic under the folders mentioned above.
